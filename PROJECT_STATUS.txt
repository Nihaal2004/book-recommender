Best Model Performance:

  - Precision@5: 0.0981 (9.81%)
  - Recall@5: 0.1534 (15.34%)
  - Strategy: 30% content-based + 70% collaborative filtering

Model Comparison:
  Content-Based:     Precision@5 = 0.014  (Baseline)

  Hybrid (α=0.3):    Precision@5 = 0.098  (Best ★)
  Hybrid (α=0.5):    Precision@5 = 0.072  (Good)
  Hybrid (α=0.7):    Precision@5 = 0.029  (Moderate)

Key Finding: Collaborative filtering dominates due to rich interaction data.
             Hybrid with low alpha (0.2-0.3) provides best results.
